兩個 SQLite 資料表欄位差異比較方法 [How to compare the differences between two SQLite table fields]

工具: https://sqlitebrowser.org/ [單純只支援sqlite]
https://www.heidisql.com/ [支援所有SQL相關]
https://portableapps.com/apps/utilities/winmerge_portable [文字檔比對工具]

sqlitebrowser
	me.db.sql
	now.db.sql

heidisql
	now.sql
	me.sql
